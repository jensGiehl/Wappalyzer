# How to build
```sudo ./run build 42```
Where ```42``` is any version number.

After that you will find in the ```build``` folder a file named ```wappalyzer_webextension.zip```. Just extract this ZIP file.


# Add to chrome extensions
* Open Chrome
* Go to ```chrome://extensions/```
* Enable *Developermode* (in the corner top right)
* Click on *Load unpacked extension*
* Choose the directory where you extract the ZIP file


# See Logs
Click on the link *Inspect views*. Now you are ready to go :)
